## Core Framework

**Virality is algorithmic amplification of content that earns specific behavioral signals** (watch-through, saves, shares, comments, profile visits) from the initial audience. The algorithm pushes content that keeps users on-platform.

### Four Scroll-Stopping Stimulus Categories

1. **Physical threat signals** — Shark, car accident, confrontation, fear. Evolutionary threat detection system overrides motor habits.
2. **Reproductive opportunity signals** — Attractive face, fitness indicators, eye contact. Dopamine anticipation response → approach motivation.
3. **Resource and status signals** — Large numbers, wealth displays. Status evaluation response → saves + profile visits.
4. **Social information signals** — Drama, gossip, interpersonal conflict. Social hierarchy navigation value.

### Platform Algorithm Weightings

| Platform | Key Signal | Optimization |
|----------|-----------|-------------|
| TikTok | 3-second retention rate | Hook must produce scroll-stop in first 3 seconds |
| Instagram Reels | Save behavior | Include explicit "save reason" — reference value |
| YouTube Shorts | Search indexing | Lead with search-intent confirm, not interruption |
| X | Specific evidence-based claims | Lead with claim, visual supports argument |
| Snapchat | Raw authenticity | Intentionally imperfect, feels handheld |

### The 4 Virality Variables

1. **Stimulus strength** — How many of the 4 attention-allocation categories does the first frame contain?
2. **Neurochemical depth** — How powerful is the response? (mild vs. intense attractiveness stimulus)
3. **Resolution quality** — How satisfying is the closure of the tension/opening created?
4. **Shareability architecture** — Does content function as social currency for the sharer?

### Volume Thesis

The RIZZ campaign: 260,900 videos → 5B views. The LooksMax AI campaign: 11,882 posts → 1.1B views. At sufficient volume × quality, viral outcomes are statistically inevitable rather than lucky.
