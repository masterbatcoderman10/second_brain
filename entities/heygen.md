## Overview
HeyGen is a commercial AI video generation platform. They released [[hyperframes]] as an open-source framework for building AI-generated launch videos using Claude Code.

## Relationship to HyperFrames
HyperFrames is HeyGen's open-source contribution to the AI agent tooling ecosystem. The framework enables creating AI-generated marketing/launch videos programmatically via Claude Code.
