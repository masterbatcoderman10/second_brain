# Vercel

Cloud platform for frontend frameworks and serverless functions. Primary deployment target for vibe-coded applications built with Next.js, and the default deployment option for [[lovable-dev]] and [[open-lovable]]. Referenced in [[hyperframes-llm-video]] as a deployment option.
