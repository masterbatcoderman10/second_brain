## Overview

Perplexica is an AI-powered search engine — the open-source project that Vane is forked from. Both are privacy-focused, self-hosted answering engines that combine LLM synthesis with web search.

## Relationship

- [[vane]] is a fork of Perplexica, maintaining a similar architecture (SearxNG + LLM synthesis with citations)
- Both projects compete in the "Perplexity alternative" space
- Vane has diverged with additional features: widgets, custom agents, image/video search
