# Gojiberry.ai

MCP server + LinkedIn execution layer that bridges AI agents (Claude) to LinkedIn's write APIs. Enables automated, personalized LinkedIn outreach campaigns launched directly from natural language prompts.

## What It Does

- Hosts an MCP server endpoint that AI agents (Claude, etc.) connect to
- Maintains LinkedIn OAuth connection on behalf of the user
- Executes campaign operations: lead enrichment, message personalization, campaign launch
- Provides real-time pipeline data back to the AI agent for analysis
