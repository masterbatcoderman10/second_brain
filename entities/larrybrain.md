# larrybrain

Skills marketplace for AI agents (OpenClaw, Hermes, Paperclip, Claude) built by Oliver Henry (@oliverhenry). One install gives the agent awareness of the entire 100+ skill catalogue.

## Overview

LarryBrain is a skills hub that differs from traditional directories (like ClawHub) in that the agent becomes instantly aware of the full catalogue after a single install — no manual browsing required.

## Key Facts

- 12,000 downloads in 50 days
- 1,000+ posts automated via LarryLoop
- Tens of millions of impressions generated through the marketing skill
- Skills download locally, fully customizable, no SaaS lock-in
- Developer ships weekly updates since January 2026
- Creators can build skills and get 50% off the subscription

## Key Skills

- **Larry Marketing Skill** — Automates X content pipeline (scans niche, writes in your voice, posts at optimal times)
- **Xcellent** — Full X analytics built into the agent; ghostwriting, viral content library, performance tracking
- **AI UGC Video Skill** — Voice note → AI video → post pipeline
- **App Template Skill** — Builds micro-apps with passive monetization logic
- **API Library** — Any developer can submit products; agent becomes aware automatically

## LarryBrain vs ClawHub

| Dimension | ClawHub | LarryBrain |
|-----------|---------|------------|
| Browse method | Manual | Ask agent |
| Catalogue size | Broader | 100+ vetted |
| Agent awareness | None | Full catalogue |
| Install per skill | Yes | One-time |
