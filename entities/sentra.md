# Sentra

Enterprise AI company building a "Company Brain" — a shared intelligence/memory layer for organizations.

## Description

Sentra is building enterprise general intelligence: a shared intelligence/memory layer that sits on all communication channels, knowledge bases, and agent traces to understand how everyone in an organization actually works as well as how work actually gets done, constructing a living world model of the entire company in near real time.
