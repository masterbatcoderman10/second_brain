# Z.ai (zai-org)

Z.ai (zai-org) is an AI research and engineering organization behind [[glm-ocr]] and other document intelligence tools. They maintain the official GLM-OCR SDK on GitHub and offer a hosted API at docs.z.ai.

## Products

- [[glm-ocr]] — multimodal OCR model (0.9B params, #1 on OmniDocBench V1.5)
- Hosted API: https://docs.z.ai/guides/vlm/glm-ocr
