# Dhruv (@dhruvtwt_)

X creator who posted NVIDIA NIM free inference thread (1,103 RT, 11,648 likes, April 2026).

## Summary

Dhruv (@dhruvtwt_) posted a thread announcing that NVIDIA offers ~80 AI models via hosted NIM APIs for free, including MiniMax M2.7, GLM 5.1, Kimi 2.5, DeepSeek 3.2, GPT-OSS-120B, and Sarvam-M. The thread went viral (1,103 RT, 11,648 likes), with replies noting the free tier is unstable and slow but useful for experimentation. Key insight: NVIDIA NIM endpoints are OpenAI-compatible, enabling drop-in replacement in existing pipelines via `base_url` substitution.

## Key Facts

- Free NVIDIA NIM inference for 80+ models
- OpenAI-compatible API (easy pipeline swap)
- Models include: MiniMax M2.7, GLM 5.1, Kimi 2.5, DeepSeek 3.2, GPT-OSS-120B, Sarvam-M
- Compatible with OpenClaude, OpenCode, Zed IDE, Hermes Agent, Cursor IDE
- Widely reported as slow/unstable by community
