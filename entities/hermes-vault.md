GitHub repository by [asimons81](https://github.com/asimons81) (Tony Simons). [[hermes-agent]] configuration repo that uses MCP protocol.

Referenced in Tony Simons' context OS article (May 2026) as his primary project where Hermes applies project-local context files (AGENTS.md, .hermes.md, CLAUDE.md, SOUL.md).

The project exemplifies the "project context" layer in [[hermes-memory-architecture]]: when Hermes enters the hermes-vault directory, it automatically picks up MCP protocol knowledge, open-source agent auth vision, and preference for iterating on existing code over fresh starts.
