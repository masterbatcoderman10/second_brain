# James Russo

HeyGen engineer and core contributor to [[hyperframes]] — HeyGen's open-source framework for AI-generated launch videos in [[claude-code]].

## Source

> [Building on top of HyperFrames](https://x.com/Rames_Jusso/status/2052465022367043830) — James Russo (@Rames_Jusso), May 7 2026

## About

James is one of the five credited HyperFrames v1.0.0 team members alongside Abhay (@AbhayZala7), Jake (@JakeFromHeyGen), Miguel (@Miguel07Code), and Vance Ingalls. His work focuses on practical applications of the HTML-for-video paradigm, demonstrating how LLMs can drive visual story generation by writing HyperFrames HTML — replacing expensive image/video gen models with code edits.

His post details building an LLM-driven visual story generator on HyperFrames at approximately **$0.02 per story** using cheap cloud models (Gemini 3 Flash), with the full pipeline deployed on Next.js + Vercel AI SDK + TTS.

## Key Demonstration

The post demonstrates that for visual products like animated explainers, dashboards-as-video, and narrated stories, you don't need photoreal video generation models (Veo, Sora, Seedance). Instead, LLMs writing code via HyperFrames achieves the same output at a fraction of the cost and with editability advantages — every tweak is a code edit, not an expensive regeneration.
