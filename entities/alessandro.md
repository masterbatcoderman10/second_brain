# Alessandro

X creator [@AlessandroLavis](https://x.com/AlessandroLavis). Covers AI-generated video advertising and creative production systems.

## Source

[Claude Opus 4.7 + Veo 3.1 Creatives Factory](https://x.com/AlessandroLavis/status/2045484341585662054) — Claims to be generating 550 animated ad videos per day using Claude Opus 4.6 (or 4.7) for concept/iteration and Google Veo 3.1 for video generation. Key claim: the system produces near-zero cost per video, production time in minutes, instant scale.

## Core Production System

- **Daily output:** 550 animated ad videos
- **Stack:** Claude Opus 4.6/4.7 + Veo 3.1
- **What AI handles:** character concepts, visual hooks, scene variations, creative iteration, performance feedback
- **Key insight:** "Creative is no longer the bottleneck. Testing speed becomes the advantage."

## Key Claim

"Weird, visual-first ads outperform polished ones." The system tests dozens/hundreds of variations daily and scales winners without touching production.
