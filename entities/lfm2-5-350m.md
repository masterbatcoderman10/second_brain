# LFM2.5-350M

**LFM2.5-350M** is Liquid AI's 350M parameter text-only language model — the LLM backbone used in [[lfm2-5-vl-450m]]. It builds on the LFM2 architecture with extended pre-training (28T tokens) and multi-stage reinforcement learning.

## Overview

LFM2.5-350M is a general-purpose model optimized for on-device deployment. Despite its small size, it achieves competitive benchmark performance against larger models through hybrid LIV convolution + GQA transformer architecture.

Also available as [[lfm2-5-vl-450m]] (vision-language variant).

## Architecture

- **Parameters**: 350M
- **Layers**: 16 total — 10 LIV convolution blocks + 6 GQA attention blocks
- **Hidden size**: 1,024
- **Attention**: 16 heads / 8 key-value heads (GQA)
- **Intermediate size**: 6,656 (SwiGLU)
- **Context length**: 32,768 tokens
- **Vocab size**: 65,536
- **Positional encoding**: RoPE, theta = 1,000,000
- **Training**: 28T tokens (extended from LFM2's 10T)
- **Architecture type**: hybrid convolution + transformer (NOT MoE)

## Inference Performance

- **AMD CPU**: 313 tok/s decode
- **Snapdragon Gen4**: 188 tok/s
- **Memory footprint**: <1GB with quantized variants

## Deployment

Supported: CPT (Unsloth/TRL), SFT (Unsloth/TRL), DPO (TRL), GRPO (Unsloth/TRL).
