# Ashpreet Bedi

X creator (@ashpreetbedi) posting on AI engineering, agentic software, and systems design. Authored the viral "Systems Engineering: Building Agentic Software That Works" thread (931 likes, 108 reposts) arguing that agentic software should be treated as a five-layer system: Agent Engineering, Data Engineering, Security Engineering, Interface Engineering, and Infrastructure Engineering.

Authored the five-layer agentic software framework and built [Dash](https://github.com/agno-agi/dash), an open-source self-learning data agent demonstrating all five layers — a Leader + Analyst + Engineer multi-agent team that writes SQL from natural language questions.

Built [Scout](https://github.com/agno-agi/scout), an open-source context agent that navigates live information sources using the Context Providers pattern. Scout connects fragmented knowledge in Slack, Google Drive, Linear, and other tools via a thin abstraction layer that prevents context pollution in multi-tool agentic systems.
