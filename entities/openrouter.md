# OpenRouter

AI API aggregation platform providing unified access to hundreds of models across major providers via a single API. Supports automatic failover, intelligent routing, and unified billing. Also offers the `@openrouter/agent` TypeScript SDK for building agents on the platform.

## Key Features
- Single API key → access to hundreds of models across providers
- Unified billing
- Automatic failover
- Intelligent routing
- Agent SDK (`@openrouter/agent`) with `callModel` primitive for multi-turn tool calling, stop conditions, streaming, state, and tool approval patterns
