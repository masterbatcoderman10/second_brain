# Zettelkasten

German for "slip box" — a note-taking and knowledge management method developed by sociologist Niklas Luhmann, who used it to write 70 books and 400 articles over 40 years. The method uses atomic notes (Zettel), each with a unique identifier, linked via explicit references to create a web of ideas.

## Core Principles

- **Atomic notes** — one idea per note, no longer than a page
- **Explicit linking** — every note links to related notes by ID
- **Unique identifiers** — each Zettel gets a unique ID enabling precise referencing
- **Emergent structure** — structure emerges from links, not top-down hierarchy
