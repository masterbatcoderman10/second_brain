# Lyra 2.0

NVIDIA's image-to-3D model that converts any 2D image into a complete 3D world in seconds. Lyra 2.0 provides full spatial understanding rather than just depth maps, representing a complete spatial understanding of scenes.

## Overview

Lyra 2.0 is NVIDIA's latest advancement in image-to-3D generation. The model takes a single 2D image as input and generates a full 3D representation with complete spatial understanding. Unlike traditional depth mapping approaches, Lyra 2.0 produces rich 3D worlds with comprehensive spatial comprehension.

## Key Capabilities

- **Image to 3D Conversion**: Transform any 2D image into a complete 3D world in seconds
- **Full Spatial Understanding**: Not just depth estimation — comprehensive 3D scene reconstruction
- **Fast Inference**: Generates 3D content in seconds, enabling rapid content creation workflows
