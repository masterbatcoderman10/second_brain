# Vadoo

## Overview

Vadoo is an AI video generation platform that provides global access to [[seedance-2-0]], positioning itself as a creator-first alternative to region-restricted Seedance access paths. Vadoo offers Seedance 2.0 in Pro, Lite, and Omni generation modes, plus a Seedance Edit workflow for edit-heavy use cases.

## Key claims from Vadoo

- **Global availability**: Seedance 2.0 available globally inside Vadoo without region-by-region restrictions or rollout politics
- **Multiple Seedance modes**: Pro / Lite / Omni for generation styles; Seedance Edit for edit-forward workflows
- **Plan tiers**: Pro (entry for serious creators) and Ultimate ($249/month for high-output creators with multiple ads/day, multiple brands, agencies)

## Seedance 2.0 access positioning

Vadoo contrasts itself with raw ByteDance API access and platform-specific restrictions by offering a hosted, globally accessible route to Seedance 2.0 generations. The platform targets creators who want to iterate fast without managing API access complexities or regional throttling.

Alternative Seedance 2.0 access routes documented in the wiki include raw ByteDance API and [[frederikfeldt-seedance-pricing]] (Danish entity raw API access at ~$0.10/sec).
