### Astra Space Tourism Landing Page (2026-04-18 thread)

[Viktor's second major prompt pack](https://x.com/viktoroddy/status/2045573887849619593) documents a complete "Astra" space tourism landing page with liquid glass morphism design. Full React/TypeScript/Vite/Tailwind/Framer Motion implementation including:

- **Liquid glass CSS:** Two variants (`.liquid-glass` and `.liquid-glass-strong`) using `backdrop-filter: blur()`, gradient borders via mask-composite, and inset shadows. Strong variant uses `blur(50px)`.
- **App structure:** Navbar → Hero (full-viewport with background video) → StartSection → FeaturesChess → Stats (HLS video) → CTA/Footer
- **Stack:** React 18, TypeScript 5, Vite, Tailwind CSS, Framer Motion, hls.js, Lucide React icons
- **BlurText component:** Word-by-word animation using IntersectionObserver with staggered motion.span animations
- **Fonts:** Instrument Serif (italic headings) + Barlow (body)

This is the most complete public prompt specification for liquid glass morphism in the vibe-coding context.
