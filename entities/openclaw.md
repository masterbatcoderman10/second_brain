# OpenClaw

Local-first AI agent with 750K GitHub stars, 800+ community skills on ClawHub. Three layers:
- **Channel layer** - how you talk to it (WhatsApp, Telegram, Slack)
- **Brain layer** - LLM it uses
- **Body layer** - tools and integrations it can reach

Better than Hermes for local file work — reads Obsidian vault, Notion exports, local PDFs, terminal output and uses those as research context.

Common community move: stack Hermes (cloud research) + OpenClaw (local research) via HermesClaw bridge.
