# Alex Cool Dev

X creator (@alexcooldev) who documented a TikTok slideshow content automation system using Claude Opus 4.7.

## Source

> [How to Automate TikTok Slideshow Content Creation with Claude Opus 4.7](https://x.com/alexcooldev/status/2044820024695947654)

## System

Stack: TikTok → SnapTik → Claude Opus 4.7 → Pinterest → Node.js

Workflow: Pinterest (endless free images) → Claude Opus 4.7 (generates slide narration/captions) → SnapTik (adds audio) → TikTok (distribution)

Note: Uses Pinterest as free image source rather than AI image generation (Cost = $0, "easy viral" vs AI-generated).
