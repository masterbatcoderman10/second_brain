# awesome-claude-code

Community-curated GitHub repository lists for the Claude Code ecosystem.

## Overview

The X creator kaize (@0x_kaize) published a widely-shared X Article cataloguing 100 GitHub repositories across 8 categories for Claude Code users. The ecosystem is evolving rapidly with new features, plugins, and community tools appearing frequently.

## Categories Covered

| Category | Description |
|---|---|
| Awesome Lists | Meta-indexes of the ecosystem (hesreallyhim, ComposioHQ, VoltAgent, ccplugins) |
| Official Anthropic | claude-code CLI, skills, SDKs, GitHub Action, MCP server, cookbook, courses |
| Skills Collections | Superpowers (148k stars), community skills, experimental labs |
| Subagents | Domain-specific agents, production-ready bundles |
| MCP Servers | github-mcp-server, playwright-mcp, postgres-mcp, context7, etc. |
| Memory/Continuity | claude-mem, claude-memory-skill, infinite-agentic-loop |
| Security/Review | claude-code-security-review, tdd-guard |
| IDE Integrations | Neovim, Emacs, VSCode plugins, Claude desktop |

## Key Repositories

- obra-superpowers (148k stars) — most popular skills library
- anthropic-claude-code (55k stars) — official CLI
- anthropic-skills (37.5k stars) — official skills library
- composiohq-awesome-claude-skills — 50+ production-ready skills
- voltagent-awesome-agent-skills — 1000+ agent skills
