# timbidefi

X creator (@timbidefi) who shared a Google Maps + Claude $10k+/week method for local service business client acquisition.

## Overview

Published an X Article detailing a four-tool workflow (Google Maps → Claude → Lovable → Higgsfield) for pre-selling website services to local businesses. The method compresses the entire pre-sales process into an afternoon by walking into the first conversation with a finished mockup and video walkthrough.

## Key Method

1. **Find leads** on Google Maps — narrow niche searches, target businesses with established offline presence but bad/no websites
2. **Claude enrichment** — batch-generate diagnosis, site brief, and cold message for each lead
3. **Lovable mockups** — build 5–8 landing page mockups for top prospects
4. **Higgsfield video** — 10-second cinematic walkthrough of each mockup
5. **Outreach** — cold message with video link + mockup URL; no AI tool mentions

## Key Claims

- 30 personalized sequences → 10–15% reply rate → 3–4 positive replies → 1–2 deals per weekend
- Monthly model: $2,000–4,000 new MRR (two weekends/month)
- One-time model: $2,500–10,000 cash per weekend
- Time to execute once dialed: 80 minutes for 30 prospects

## Tools

- Claude Code — for writing outreach copy and site briefs
- Lovable — for building hosted landing page mockups
- Higgsfield — for generating demo videos
- Google Maps — for lead discovery
