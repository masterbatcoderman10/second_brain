# refero-styles

[[refero-styles]] is a dataset project that collects 2,000 DESIGN.md files from top products worldwide. The dataset is structured for AI models to read and learn good design principles including colors, typography, spacing, and layouts.

This project directly addresses the problem of AI agents producing poor UIs by exposing them to curated examples of quality design documentation. The resource is freely available and intended to improve design generation capabilities in AI systems.
