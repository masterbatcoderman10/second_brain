# 10 Prompts That Save 40+ Hours a Month

> Source: X (Twitter) article
> Topic: Practical prompts for Claude Code, content creation, SaaS building, and productivity
> Author: Unknown (shared via private GitHub folder of engineers/founders)

## Prompt 1: Writing Like a Human
Rewrite this. Remove filler phrases.
Vary sentence length. Add contractions.
Use specific details. Highlight every
change and explain why.
Flag anything still robotic: [TEXT]

## Prompt 2: Hook Generator
Generate 20 scroll-stopping hooks for [TOPIC] using:
curiosity gap, bold claim, statistic,
story opener, controversial take.
Rate each 1-10 on virality with a one-line justification.

## Prompt 3: Content Brief Prompt (SEO & Growth)
Generate a full content brief for [KEYWORD].
Include: search intent, word count,
H2/H3 structure, competitor gaps,
meta description, internal linking targets,
featured snippet strategy, schema markup.

## Prompt 4: Idea Validation Prompt (SaaS Building)
Validate this idea with first-principles analysis.
Market size, competition, defensibility,
build/no-build verdict.
MVP scope with timeline.
The 3 riskiest assumptions to test first: [IDEA]

## Prompt 5: Objection Handler (Marketing & Sales)
List 15 objections people have before buying [PRODUCT].
For each: one-liner counter for copy.
Categorize by type (price, trust, timing, competition).
Suggest exactly where to place each counter on the page.

## Prompt 6: Brutal Code Review (Claude Code & CLI)
Do a git diff.
Pretend you're a senior dev who HATES this implementation.
Tear it apart -- what would you criticize,
refactor, or reject in a PR review?

## Prompt 7: Task Prioritization (Productivity & Systems)
Turn this task list into a 7-day execution plan.
Flag what I can delegate.
Flag what I can automate.
Identify the one task that unblocks everything else.
Add buffer time: [TASKS]

## Prompt 8: Production Bug Prompt (Debugging & Refactoring)
Walk me through a systematic debug process with 10 diagnostic steps.
Include an environment diff checklist,
a logging strategy, and how to set up monitoring
to catch this next time.
Error: [ERROR]

## Prompt 9: Above-the-Fold Copy (Landing Pages & Copy)
Write above-the-fold copy for [PRODUCT]
that communicates value in 3 seconds.
Headline, subheadline, CTA text, social proof line.
3 variations: benefit-led, curiosity-led, social-proof-led.
Hero image direction for each.

## Prompt 10: Digital Product Generator (Monetisation & Scale)
Generate 10 digital product ideas I can build
from my expertise in [SKILL].
Include pricing, format, estimated build time.
Rank by revenue vs effort.
Identify which products cross-sell with each other.

## Author's Results (6 weeks)
- Token usage: down 60% ($180/month -> $40/month)
- Output quality: up
- Time spent on rewrites: almost zero
- First drafts shipped: 2/week -> 7/week
- 3-hour tasks: done in 25 minutes
- Saved: $140/month, 40+ hours/month
