## What Coolify Is

An open-source and self-hostable alternative to Heroku / Netlify / Vercel / etc. It helps you manage servers, applications, and databases on your own hardware — you only need an SSH connection. Supports VPS, Bare Metal, Raspberry PIs, and anything else.

Key pitch: the ease of a cloud with your own servers, no vendor lock-in. All configurations are saved to the server itself, so if you stop using Coolify, resources keep running.

## Features

- **280+ one-click services** — databases, queues, full-stack apps, static sites
- **Self-hosted PaaS** — SSH-only server access required; no agents installed on local machines
- **Server management** — manage multiple servers from one Coolify instance
- **Installation:** `curl -fsSL https://cdn.coollabs.io/coolify/install.sh | bash`
- **Paid cloud option:** app.coolify.io for those who don't want to self-host
- **Sponsors:** Huge sponsor list (MVPS, SerpAPI, ScreenshotOne, Hetzner, Hostinger, etc.) and 50+ big/small sponsors
