Quoted Tweet by @kylejeong — "Autobrowse: The Mythos moment for Browser Agents is here"

Browser agents have an amnesia problem. They re-discover every site from scratch on every run, paying the full discovery tax forever. Autobrowse fixes that by letting an agent iterate on a real task until it converges, then graduating the winning approach into a durable, reusable skill.

How it works:
1. Objective — hand the agent a real task on a real site
2. Run — agent attempts the task end to end
3. Study — agent reads its own trace, identifies what stalled/guessed/wasted tokens
4. Strategy — run's outer loop maintains strategy.md scratchpad
5. Iterate — refine based on notes, drop steps that didn't pull weight
6. Converge — short-circuit when consecutive iterations stop yielding improvements
7. Graduate — write SKILL.md plus helper files into the public skills repo

The output is a small, readable markdown file — no transcript, no vector embeddings, no screenshot reel.

Example: Craigslist scrape with generic agent loop: $0.22 / 71 seconds. Graduated Autobrowse skill: $0.12 / 27 seconds.
