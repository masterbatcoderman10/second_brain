# Open Higgsfield AI — Open-Source Alternative to Higgsfield AI

**Hosted:** muapi.ai/open-higgsfield-ai

The free, open-source alternative to Higgsfield AI. Generate AI images and videos using 200+ state-of-the-art models — without the closed ecosystem or subscription fees.

## Why Open Higgsfield AI
- Free & open-source — no subscription, no vendor lock-in
- Self-hosted — your data stays on your machine
- 200+ models — text-to-image, image-to-image, text-to-video, image-to-video, lip sync
- Multi-image input — feed up to 14 reference images into compatible models
- Lip Sync Studio — animate portraits or sync lips to any audio with 9 dedicated models
- Extensible — add your own models, modify the UI, build on top of it

## Powered by Muapi.ai
Models: Flux, Nano Banana, Midjourney, Kling, Sora, Veo, Seedream, Infinite Talk, LTX Lipsync, Wan 2.2, and more.

## Studios

### Image Studio
- Text-to-image: 50+ models
- Image-to-image: 55+ models
- Auto-switches model set based on whether reference image is provided
- Quality and resolution controls
- Multi-image input: up to 14 reference images for edit models (Nano Banana 2 Edit, Flux Kontext Dev, GPT-4o Edit)
- Multi-select picker with order badges, batch upload

### Video Studio
- Text-to-video: 40+ models
- Image-to-video: 60+ models
- Same intelligent mode switching as Image Studio

### Lip Sync Studio
- 9 dedicated models across two modes:
  - Portrait image + audio → talking video
  - Video + audio → lipsync video

### Cinema Studio
- Higgsfield AI-style interface for photorealistic cinematic shots
- Pro camera controls: Lens, Focal Length, Aperture
- "Infinite Budget" cinema workflow

## Other Features
- Upload History — reference images stored locally, reusable across sessions
- Smart Controls — dynamic aspect ratio, resolution/quality, duration pickers adapt per model
- Generation History — browse, revisit, download all past generations (browser storage)
- Image & Video Download — one-click, full resolution
- API Key Management — stored in browser localStorage, never sent except to Muapi
- Responsive Design — desktop + mobile, dark glassmorphism UI

## Distribution
- Hosted web app: muapi.ai/open-higgsfield-ai
- Desktop apps: macOS (Apple Silicon + Intel), Windows (x64 + ARM64)
- macOS requires xattr -cr for Gatekeeper bypass (not notarized)
- Windows requires SmartScreen "Run anyway" (not code-signed)
