# FiveM GTA UGC

The FiveM modding platform as a pre-positioning opportunity for the GTA 6 creator economy. Thesis by [[0xdepressionn]]: Rockstar's 2023 FiveM acquisition was a talent hire to build GTA 6's UGC infrastructure, and AI coding agents can be used to build FiveM Lua scripts for resale on the Cfx Marketplace.

## Key Claims

- **GTA 6 launches November 19, 2026** — $7.6B lifetime revenue projected by Take-Two
- **Rockstar acquired FiveM team in 2023**, relaunched paid marketplace January 2026
- **Rockstar is hiring 4 Creator Platform roles** for GTA 6 infrastructure
- **FiveM already has paying customers** — Lua scripts selling on Cfx Marketplace, RP servers with 200+ paying members at $20/month
- **If 1% of GTA 6 players spend $10/month on UGC** — $240M annual creator economy

## The Opportunity

FiveM scripting market is **under the radar for AI creators**:
- Existing buyers, existing payment infrastructure
- Low competition from AI-native content
- Each completed script = proof of work for GTA 6 creator program

## AI Coding Agent Workflow

Claude Code can generate FiveM Lua scripts from natural language prompts:

```
I want to build a FiveM script to sell on the Cfx Marketplace.
[describe script type]. Build a complete, working script.
```
