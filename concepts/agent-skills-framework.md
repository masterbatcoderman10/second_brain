# agent-skills-framework

ridark's framework for structuring AI agent skills. Core principles: keep skills concise (~50 tokens), use `references/` directory for depth, set degrees of freedom, test across Haiku/Sonnet/Opus, delegate creation to AI. Contrasts with [[skillify]]'s failure-conversion approach — focus on concise skill authorship rather than failure-driven skill creation.
