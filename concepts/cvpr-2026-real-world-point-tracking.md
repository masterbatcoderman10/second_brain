# Real-World Point Tracking with Verifier-Guided Pseudo-Labeling

Point tracking system for real-world objects using verifier-guided pseudo-labeling.

## Overview

Addresses point tracking in real-world scenarios where ground truth annotations are scarce. Uses a verifier network to guide the generation of pseudo-labels, improving tracking robustness in challenging conditions. Marked as a highlighted paper.
