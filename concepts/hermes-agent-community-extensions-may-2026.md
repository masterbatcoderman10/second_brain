# Hermes Agent Community Extensions (May 2026)

Five new community-built Hermes Agent extensions (all publicly verified and accessible) that extend Hermes's core loop into backup, visualization, creative writing, security auditing, and music composition.

## The Five Extensions

### 1. hermes-agent-backup-skill

GitHub: t.co/tz3MD0b1do

Nightly automatic backup of Hermes memory, skills, and configuration into a private GitHub repository. Solves the "Agent 失忆时代" (Agent Amnesia Era) — agents losing memory across sessions.

**Key claim:** "Agent 失忆时代，结束了！" (The Agent Amnesia Era is over!)

### 2. telepath

GitHub: t.co/zasAaRSt4F

Hackathon project that feeds Hermes memory into Kimi K2, converting vague requirements into polished charts, Mermaid diagrams, and slide presentations — then writes results back into the memory store.

**Key claim:** Memory visualization ceiling reached (😂)

### 3. hermes-novel-generator

GitHub: t.co/qc4BpliRlN

Multi-agent fiction pipeline: Orchestrator + character agents + audit + polishing → directly outputs EPUB files.

**Key claim:** "Agent 也要当小说家了！" (Agents also become novelists!)

### 4. psy-core

GitHub: t.co/OyqOwTmSRR

Tamper-proof memory audit plugin using hash chains + HMAC signatures. Every operation is traceable and verifiable.

**Key claim:** Enterprise teams finally dare to put Hermes into production.

### 5. music-skill

GitHub: t.co/1xsDTpzXk7

MIDI analysis, modification, and FluidSynth rendering skill. Agents become composition masters.

**Key claim:** Musicians rejoice — AI also creates!

## Why These Extensions Matter

All five projects share a common thread: they all use Hermes's 底层循环 (core loop) as DNA, with the community layering on:

- **Backup/restore** — solve memory loss across sessions
- **Visualization** — make memory state inspectable
- **Creative output** — extend beyond code to EPUB
- **Security** — hash chains for audit trails
- **Music** — MIDI/FluidSynth composition

The ecosystem velocity is described as "肉眼可见" (visibly accelerating).
