# instagram-ugc-system

A system for building viral Instagram presence using AI-generated UGC (User Generated Content) video, anchored by Seedance 2.0 and multi-account portfolio strategies.

## Overview

Two overlapping systems documented by @adriansolarzz and @afwLaur for using AI video generation to produce viral Instagram (and TikTok) content at scale, driving affiliate and brand revenue.

## Core Components

### 1. Content Generation (Seedance 2.0)
- Passes the "real person" quality threshold in first 2 seconds
- Native audio-visual sync in single pass (no lip-sync drift)
- Skin rendering with realistic texture and subsurface scattering
- Natural human physics in motion

### 2. Script System
- 3-input stack: reference ad transcript + product brief + audience insight
- Claude used for hook generation (20+ variations per message)
- Proven ad frameworks reverse-engineered and replicated

### 3. Distribution Architecture
- Multi-account portfolio: 10-50 niche-specific accounts
- 2-4 week account warmup protocol before deployment
- Collaboration posts for engagement multiplication
- DM shares as highest-value algorithmic signal

### 4. Conversion Funnel
- Comment funnel with keyword micro-commitment → DM sequence
- Link-in-bio as backup capture layer
- Multi-touch sequences (3-5 touchpoints before purchase)

## Key Formats That Work

1. **Before/after transitions** — 4.4M likes record; emotional projection drives shares
2. **Slideshow story arcs** — 27 slides, 80% completion rate; dwell time signals
3. **Viral copy with product twist** — Copy proven viral structure with AI-generated character

## Metrics Reported

- @adriansolarzz: 2.6M organic views in 48 hours; $800K+ total generated
- @afwLaur: $500-$1.5K/day; $30K/month target
- Best format: before/after (4.4M likes single post)
