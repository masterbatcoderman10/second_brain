# 60-claude-cowork-moves

A catalog of 60 specific Claude Cowork commands, skills, and workflows organized across six layers of increasing sophistication. Authored by [[shmidt]] in a May 5, 2026 X Article (178 likes, 13 RTs).

## Overview

The article argues that most users never scratch past the first layer of Cowork, and that stacking all six layers turns Cowork from "an AI tool I open sometimes" into an autonomous operating system running the boring 70% of work in the background.

## The Six Layers

### Layer 1: Slash Commands (12 commands)

Beginner-accessible commands for session control and task management:
- **/schedule** — Background task scheduling with laptop closed support
- **/loop** — Interval polling inside current session (e.g., build monitoring)
- **/plan** — Forces step-by-step planning before execution (recommended for tasks touching 3+ files)
- **/compact** — Compresses long conversations, saves 40–60% context window
- **/clear** — Nuclear thread reset for context pollution
- **/resume** — Pick up previous conversations
- **/rename** — Auto-renames threads for searchable sidebar
- **/cost** — Shows estimated token cost before running tasks
- **/memory** — Shows which memory files and context are loaded
- **/doctor** — Diagnostic command listing connected apps, skills, commands
- **/voice** — Push-to-talk dictation (Pro feature)
- **/agents** — Spawn project-specific sub-agents (Pro feature)

### Layer 2: Custom Skills (12 skills)

Reusable mini-programs loaded automatically when relevant. The article recommends installing from the 2,300+ free skills marketplace:
- Brand voice skill (5–10 articles → consistent tone)
- Email triage skill (personal rules encoding)
- Meeting transcript skill (raw transcript → structured notes)
- PDF report generator skill
- Code review skill (team standards encoding)
- Customer support response skill
- Proposal generator skill
- Investor update skill
- Contract review skill
- Hiring screen skill
- Stakeholder communication skill (4 output formats)
- Pricing skill (3-tier generation with rationale)

### Layer 3: File System Power Moves (9 moves)

- Screenshot library organizer (OCR + categorization)
- Smart attachment processor (email grouping, invoice renaming)
- Code repo cleanup (dead branches, unused dependencies)
- PDF metadata extractor (research database creation)
- Photo library curator (duplicate detection, quality selection)
- Voice note → article pipeline (transcripts to publication drafts)
- Custom skill folder setup (~/skills with SKILL.md per skill)
- Git history miner (quarterly engineering self-review)
- Backup audit with intelligence (change detection, deletion flagging)

### Layer 4: Connector Workflows (12 workflows)

Integration chains replacing entire job functions:
- Calendar conflict resolver
- Slack thread → executive brief
- Gmail → CRM logger
- Drive → data warehouse pipeline
- Notion → Linear ticket pipeline
- Multi-source weekly status report
- Cross-platform context retrieval (meeting prep)
- Salesforce pipeline hygiene
- GitHub PR auto-review
- Stripe → bookkeeping reconciliation
- Customer churn early warning
- Investor pipeline tracker

### Layer 5: Automations That Run While You Sleep (8 automations)

Scheduled background tasks:
- Daily 7am inbox processor
- Sunday week-ahead brief
- Daily metric snapshot
- Weekly content recycler (1 article → 14 distribution assets)
- Monthly competitor scan
- Quarterly OKR self-review
- Annual highlights compiler
- Birthday & relationship maintenance

### Layer 6: Pro Patterns (7 patterns)

- Sub-agent parallelization (parallel research/memo/slide drafting)
- Connector chains for compound output (4 tools, 1 prompt)
- Skill stacking for compounding leverage (skills compose like Lego)
- Plan mode for token discipline (see full budget upfront)
- The .claude/ folder convention (project context auto-loading)
- Off-peak scheduling for 2× throughput (Anthropic doubled limits March 2026)
- Persistent Dispatch threads across devices (thread survives device switches)

## Key Pro Tips

1. Treat Cowork like a junior employee, not a search box — context and examples matter
2. Build your skill library in the first month, not as you go
3. /plan is free insurance — 3 seconds of typing vs potential hours of correction
4. Schedule the boring stuff first — foundation for time savings
5. The .claude/ folder is your new resume — hiring managers check it in 2026
