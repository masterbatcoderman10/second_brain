# marketingskills-repo

A free GitHub repo ([coreyhaines31/marketingskills](https://github.com/coreyhaines31/marketingskills)) that packages a full marketing agency into AI agent-readable markdown files. Covers 139 growth tactics, 12 programmatic SEO playbooks, and frameworks for copywriting, CRO, pricing, and A/B testing. Positioned as a $0 replacement for a $10,000/month marketing agency.

## Core Insight: Product Marketing Context File

Every skill in the repo starts with `product-marketing-context.md` — a file the agent reads before any other task. It contains ICP, personas, pain points, competitive landscape, objections, and verbatim customer language. Without this file the agent writes generic output; with it the agent knows the customer's exact words.

## Switching Forces Framework

The central conceptual contribution: four forces that explain why customers switch or don't:
- **Push** — what's frustrating enough about the current solution to trigger a search
- **Pull** — what's attractive about the new solution
- **Habit** — what keeps people stuck even when unhappy
- **Anxiety** — what fears prevent switching even when desired

Most marketing addresses only Pull. The Switching Forces framework requires addressing all four.

## Key Frameworks

### Copywriting
Four non-negotiable principles: clarity over cleverness, benefits over features, specificity over vagueness, customer language over company language. Headline and CTA formulas provided.

### CRO Audit (7-Question Framework)
Value proposition clarity → headline effectiveness → CTA placement → visual hierarchy → trust signals → objection handling → friction detection.

### A/B Testing
Structured hypothesis format: `Because [data], we believe [change] will cause [outcome]`. ICE scoring (Impact + Confidence + Ease / 3) for prioritization.

### Pricing
Three axes: packaging, pricing metric, price point. Anchoring, decoy effect, charm pricing, round pricing principles.

### AI SEO
Three pillars: Structure (extractable blocks), Authority (citable content), Presence (AI platform distribution). Key tactic: `/pricing.md` for machine-readable pricing.

### Programmatic SEO — 12 Playbooks
Templates, Curation, Comparisons, Examples, Locations, Personas, Integrations, Glossary, Translations, Directory, Profiles. Rule: 100 strong pages beats 10,000 thin pages.
