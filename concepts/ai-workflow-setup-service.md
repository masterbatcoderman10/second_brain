# AI Workflow Setup Service

Selling AI workflow automation setup as a freelance service for small and medium businesses (SMBs).

## Overview

SMBs (~76%) actively want AI integration but lack the technical knowledge to implement it. This creates a service opportunity: learn the toolset, identify SMB pain points, and build automations that eliminate repetitive manual tasks.

**Core skill:** Audit a small business's repetitive tasks, identify the 2–3 costing most time, build AI-powered workflows that eliminate them.

**Entry pricing:** $500/setup session → $500–5,000/month retainers once established.

## Tool Stack

| Layer | Tools | Role |
|-------|-------|------|
| Automation | Make.com, Zapier | Connect apps, trigger actions |
| Intelligence | Claude / ChatGPT | Transform unstructured input into structured output |
| Integration | API connections, webhooks | Connect to CRM, email, forms, Slack |

## Weekend Learning Path

1. **Saturday morning:** Make.com/Zapier tutorials (3 hrs) — understand scenarios, modules, data flow
2. **Saturday afternoon:** Build a real workflow for yourself — breaking/fixing IS the learning
3. **Sunday:** Study SMB pain points and practice translating technical capability into business value

## The Sales Model

- **Free audit call** (1 hr): identify top time-consuming manual tasks + show how automation solves them
- **$500 setup fee:** 1–3 connected automations solving a defined recurring problem
- **Monthly retainer:** $500–5,000/month for maintenance and expansion

### Realistic $500 Delivery

Example: Lead form → automation creates CRM record + sends personalized Claude-drafted email to lead + sends Slack summary to owner. Runs automatically, eliminates ~2 hrs/week.

**Math clients understand:** 2 employees × 3 hrs/week × $50/hr = ~$300/month in labor on a problem a $500 automation permanently solves.

## Progression Path

1. **First week:** $500–1,000 from 1–2 converted audit calls
2. **Months 1–3:** Monthly retainers from first clients ($500–2,000/month per client)
3. **Months 3–6:** Productize — certain business types (dental, real estate, law) have near-identical workflows; package and sell fixed-price once built 3x
