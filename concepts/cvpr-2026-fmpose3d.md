# FMPose3D: Monocular 3D Pose Estimation via Flow Matching

Monocular 3D human pose estimation using flow matching as the underlying generation paradigm.

## Overview

FMPose3D applies flow matching — a recent generative framework — to the problem of estimating 3D human pose from a single RGB image. Flow matching provides a differentiable path from noise to pose, enabling efficient sampling-based pose reconstruction.
