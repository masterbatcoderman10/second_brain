## Summary

An 8,000+ character step-by-step guide by [[explorax_]] on building high-motion animated business websites using Claude Code + Framer Motion (formerly Motion), with a "nano banana pro" (Gemini 3 Pro image model) workflow. Author presents it as a profitable business model ($10k agency-rate sites) rather than just a tutorial.

**Stack:**
- Claude Code — AI coding agent ($20/mo Claude Pro)
- Framer Motion (Motion) — open-source React animation library (free)
- Nano Banana Pro — Gemini 3 Pro image model for hero assets (free tier on AI Studio)
- Next.js + TypeScript + Tailwind as the framework

**Key technique:** Generate exploded-view product images with Gemini, turn them into a short video (Runway/Kling), feed to Claude Code which reads the video and builds a scroll-driven landing page with Framer Motion animations matching the video's motion arc.

**Polishing tip:** Custom cubic-bezier easing `[0.22, 1, 0.36, 1]` and staggered `whileInView` reveals for Apple-tier feel.
