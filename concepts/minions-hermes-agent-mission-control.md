# Minions - Hermes Agent Mission Control

## Definition

An open-source mission control tool for Hermes Agent that automatically supervises every running task.

## Key Points

- Solves the problem of managing multiple AI agents in parallel
- Addresses issues with cron jobs failing silently and tasks getting blocked
- Reduces time spent babysitting AI agent tasks
- Open-source project by @an_engineer_log

## Context

The author highlights that while a single AI agent works great for one task, managing 20 in parallel presents significant challenges. Minions was built to solve this supervision problem.
