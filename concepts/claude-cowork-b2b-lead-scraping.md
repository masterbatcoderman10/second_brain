# Claude Cowork B2B Lead Scraping

Using Claude Opus 4.7 via the Claude Cowork Chrome extension to scrape untapped B2B lead databases as an alternative to saturated platforms like Apollo.io.

## Core Problem

Apollo.io (250M contacts) is oversaturated — every cold email agency uses the same lists. Reply rates are 2% because prospects have been pitched 50+ times this month. Deliverability suffers because everyone hits the same domains.

## The Alternative: Untapped Databases

The argument is to find leads in platforms competitors haven't automated:

| Platform | Signal Scraped |
|---|---|
| LinkedIn | Job changes, promotions, post engagement, problem discussions |
| Twitter | Active pain-point discussions, competitor engagement, Space participants |
| Reddit | Problem-seekers in relevant subreddits |
| Quora | Question-askers in relevant topics |
| Crunchbase | Recently funded companies, new executive hires |
| Product Hunt | New tool launches, maker profiles |
| Discord/Slack | Community members, problem discussions |
| Google Maps | Local business profiles, review patterns |
| Universities | Faculty directories, research labs, incubator founders |
| Events | Speaker/sponsor/attendee lists |

## Framework

1. Identify where ICP congregates online
2. Feed platforms to Claude Cowork with specific signal criteria
3. Claude extracts: name, title, company, contact info, behavioral context
4. Personalize outreach based on scraped context

## Why Opus 4.7 Changes Things

Opus 4.7's visual accuracy upgrade enables scraping platforms with:
- Profile screenshots (LinkedIn, Instagram)
- Engagement pattern analysis
- Bio formatting interpretation
- Visual content on otherwise hard-to-scrape platforms
