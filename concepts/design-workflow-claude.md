# design-workflow-claude

A workflow for using 400+ design systems with Claude to generate design templates and DESIGN.md files.

## Overview

Described by @MengTo: pull from a library of 400+ design systems, feed a design request to Claude, and Claude generates the rest — producing either a DESIGN.md file or full HTML templates.

## Key Mechanics

- 400+ design systems as source reference library
- Claude used as the generation engine to complete/compose the design
- Supports both DESIGN.md (spec format) and HTML output
- Focus on speed and iteration: "super simple workflow"
