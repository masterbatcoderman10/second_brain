# AI Cartoon Character UGC System

System for building and monetizing AI-generated cartoon character content on TikTok US, documented by [[babeclipping]]. The approach uses SpongeBob SquarePants characters as the anchor IP, generates scripts via [[claude-code]], creates images with Nano Banana Pro, and animates with [[seedance-2-0]].

## Core Process

1. **Idea**: AI-generated 3D cartoon animations of well-known characters (SpongeBob) in adult/controversial situations targeting US TikTok audience
2. **Script**: Claude writes viral scripts using competitor transcript analysis; hook from frame 1, reaction/escalation, chaos peak, twist/payoff, punchline ending
3. **Image Generation**: Nano Banana Pro generates character reference images on white background, then scene starting frames
4. **Animation**: Seedance 1.5 Pro animates starting frames with baked-in voiceover and sound effects
5. **Editing**: CapCut for trimming, auto-captions (bold white/yellow text, black outline, one word at a time)
6. **Monetization**: TikTok Creator Rewards Program (CRP) + brand campaigns via affiliate networks

## Key Tools

- [[claude-code]] — Script generation via competitor transcript analysis
- [[seedance-2-0]] — Video animation with baked-in voiceover
- Nano Banana Pro — Image generation
- AffiliateNetwork.com — Brand campaign monetization ($1-3 per 1K views)

## Monetization

- TikTok CRP: ~$0.10/1K views (rugpull risk — TikTok can revoke without warning)
- Brand campaigns: $1-3/1K views, paid directly (no rugpull risk)
- Creators using brand campaigns reportedly making $40K+/month

## Risks

- TikTok rugpull: CRP accounts being pulled without warning, earnings seized
- US-only audience required for premium CPM rates (requires virtual US setup)
- Platform dependency — should not be primary income stream
