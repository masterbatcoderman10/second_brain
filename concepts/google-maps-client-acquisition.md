# Google Maps Client Acquisition

5-step method for finding and converting local business clients using Google Maps, Instant Data Scraper, and Claude Code — no ads, no paid lead lists.

## Summary

Practical guide for finding unlimited paying clients using Google Maps and a free scraper tool. The method focuses on identifying local businesses with "holes" in their online presence (low reviews, no website, unresponded reviews, missing listing data) and using cold outreach (calls first, email fallback) to book meetings. Claude Code handles CSV filtering and email personalization at scale.

## Key Concepts

- **4 prospect signals**: low review count (<30), no website linked, no responses to reviews, missing listing basics (photos, hours, description). 2+ signals = prospect.
- **Instant Data Scraper**: Free Chrome extension that extracts business data from Google Maps into CSV (names, ratings, phone numbers, websites).
- **Claude Code filtering**: Score each business 0–4 against the 4 signals, fetch websites to check review response, output qualified prospects sorted by score.
- **Cold call 10-second opener**: Lead with a specific observation about their listing (not a pitch), give them an out, end with a question.
- **Email fallback**: Claude Code writes personalized cold emails per prospect using signal_notes for specificity.
- **3 objections**: "not interested" (agree + reframe), "send me an email" (get small commitment first), "how much?" (defer to follow-up call).
- **Walkthrough booking**: "15-minute walkthrough" not "meeting", specific time slot, calendar invite within 5 minutes.
