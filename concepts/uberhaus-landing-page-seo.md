## End Goal

Capture organic leads for UberHaus interior design/finishing services in Bangalore via a search-optimized landing page. If SEO performs, replicate the model across other local service niches.

## How Monetizable

- Leads → conversions → revenue for UberHaus directly
- Replication model: prove it works for Waleed → sell the same playbook to other interior designers in Bangalore and beyond
- If replication works: white-label or template-based delivery (low marginal cost per new niche)
- SEO timeline is long (3–6 months minimum); not a fast revenue vector

## Vector

AI-generated content (primary), AI-generated ads (supporting)

## Target Market

- Primary: Bangalore homeowners, landlords, small business owners seeking interior design and finishing services
- Secondary: Other interior designers/decorators in Bangalore who have zero digital presence and can replicate the landing page model

## SRS: Product Requirements

### Core Features

- [ ] Responsive landing page (mobile-first design)
- [ ] UberHaus logo and brand identity
- [ ] Services section: list of interior finishing offerings (modular kitchen, wardrobes, false ceiling, painting, flooring, etc.)
- [ ] Project portfolio: photo gallery of completed work with room/project type labels
- [ ] Before/after showcase if available
- [ ] Contact form: name, phone, email, project type, approximate budget, message
- [ ] WhatsApp CTA button (floating, persistent on mobile)
- [ ] Google Business Profile integration / embed
- [ ] Testimonials section (if available; placeholder if not)

### SEO Requirements

- [ ] Local SEO: Bangalore as primary geo-target
- [ ] Service pages for each offering (modular kitchen, wardrobes, etc.) — prevents thin content
- [ ] On-page schema: LocalBusiness schema, Service schema, Image schema
- [ ] Google Business Profile optimization (description, categories, photos)
- [ ] Page speed: < 3s load on mobile, Lighthouse score > 85
- [ ] Core Web Vitals compliance (LCP, CLS, INP)
- [ ] Internal linking between service pages and portfolio

### Optional / Future

- [ ] Blog section for interior design tips (content marketing layer)
- [ ] Inquiry tracking: store leads in DB, not just email
- [ ] Quote request form with project type auto-population
